# Agent Plugins

This repository serves as a dedicated collection of domain-specific plugins for the agent harness.

## Purpose

The primary goal of this repository is to capture and formalize my personal workflows, best practices, and implicit rules (the "unwritten rules") across various domains and specific scenarios. By translating these implicit methodologies into concrete, executable plugins, the agent harness can understand and adopt my exact workflows.

Consequently, when I face similar tasks in the future, the agent harness will be able to leverage these plugins to autonomously assist me, ensuring consistency and efficiency based on my personal standards.
